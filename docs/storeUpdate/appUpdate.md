# 업데이트 표시가 안뜨는 경우

    앱이 빠르게 실행되기 위해 저장하는 캐시 데이터로 인해 업데이트 된 정보를 바로 불러오지 못하는 현상입니다. 


구글 플레이 스토어 캐시 삭제
	1.	설정 > 앱으로 이동합니다.
	2.	Google Play 스토어를 찾아 선택합니다.
	3.	저장 공간 및 캐시를 선택한 후 캐시 삭제를 실행합니다.
