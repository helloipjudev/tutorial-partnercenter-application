    2.	기본 정보 입력
	•	이메일: 사용 가능한 이메일 주소입력하여 인증번호를 받기를 클릭 후 입력한 메일로 받은 인증번호를 입력해주세요.
	•	비밀번호: 안전한 비밀번호를 생성하고, 동일한 비밀번호를 한 번 더 입력해 확인합니다.
